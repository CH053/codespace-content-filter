# Codespace Content Filter README

סינון תמונות של סביבת פיתוח גיטהאב

## פיתוח חדש!

לא מאפשר תצוגת והורדה תמונות או וידאו מהשרת של גיטהאב.
מאפשר העלאת קבצים רגיל.

לא עושה שום דבר בסביבה מקומית.

לא מאיט את הביצועים של השרת.

## התקנה

1. הורד את קובץ ה-`.vsix` מדף ה-Releases.
2. פתח את VS Code.
3. העלה את הקובץ לשרת גיטהאב.
4. לחץ על התקן קובץ `.vsix`.
5. יופיע חלון שיבקש הפעלה מחדש יש לאשר.
6. זהו הכל מוכן.
