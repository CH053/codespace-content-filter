# 1568741230 README

סינון נטפרי על תמונות ב-VS Code. 
ההרחבה מוודאת שכל תמונה שנפתחת בעורך עוברת דרך שרת סינון לפני הצגתה.

## התקנה

1. הורד את קובץ ה-`.vsix` מדף ה-Releases.
2. פתח את VS Code.
3. לחץ על קיצור הדרך `Ctrl+Shift+X` כדי לפתוח את חלונית ההרחבות.
4. לחץ על תפריט שלוש הנקודות (...) בפינה העליונה.
5. בחר ב-**Install from VSIX...** ובחר בקובץ שהורדת.
